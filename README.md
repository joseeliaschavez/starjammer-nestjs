# starjammer-nestjs

## Description

An example Node.js/Nest.js backend application capturing microservice best practices.

This app uses Nest.js, a progressive Node.js framework for building efficient and scalable server-side applications.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
